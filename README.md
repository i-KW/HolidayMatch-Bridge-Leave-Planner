# HolidayMatch · Bridge Leave Planner 🗺️

跨国朋友/家庭一起规划假期？不同国家的日历、假期、调休拼在一起眼花缭乱。
这个仓库把两两国家的工作和节假日日历对照做成了一份 HTML，AI 还顺手算好了拼假方案（Bridge Leave）。

## 已收录

| 文件 | 国家 | 说明 |
|------|------|------|
| `China＆Ireland_HolidayMatch_Calendar.html` | 🇨🇳 中国 · 🇮🇪 爱尔兰 | 含调休补班 + 拼假攻略 |
| `UK＆Spain_HolidayMatch_Calendar.html` | 🇬🇧 英国 · 🇪🇸 西班牙 | English + Los Puentes 桥假 |

## 怎么用

用浏览器打开对应的 HTML 就行。一个页面里：
- 两个人各自的公众/银行假日（颜色区分）
- 两人共同放假的日期（紫色高亮）
- 全年假日汇总表
- AI 生成的拼假方案 —— 请几天年假能连成最长假期

适合直接丢到群里或家庭共享文档里一起看。

## 风格

纯 HTML + CSS，单文件，不需要装任何东西。打印友好，也可以直接当网页看。
