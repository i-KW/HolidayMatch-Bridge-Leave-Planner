# HolidayMatch · Bridge Leave Planner 🗺️
跨国朋友/家庭一起规划假期？各自去找放假通知很麻烦？群里一个个日期打字各自的假期费时间还对不上号？不同国家的日历、假期、调休列在一起没有日历让人眼花缭乱？
这个项目致力于把多个国家的工作日和节假日按日历格式对照做成了一份 HTML，AI 还顺手算好了拼假方案（Bridge Leave Guide）。

## 怎么用

用浏览器打开对应的 HTML 就行。一个页面里：
- 两个国家各自的公众/银行假日（颜色区分）
- 两个国家共同放假的日期（紫色高亮）
- 全年假日汇总表（Bridge Leave Guide）
- AI 生成的拼假方案 —— 请几天年假能连成最长假期

适合直接丢到群里或家庭共享文档里一起看。

## 技术栈

纯 HTML + CSS，不需要装任何东西，直接浏览器打开对应文件即可。打印友好（浏览器右键自助打印），也可以直接当网页看。
